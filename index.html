TubeFold

Turn any YouTube video into a clean Markdown summary — using your existing codex or claude CLI subscription. No API key, no per-token billing.

$ brew install --cask tubefold/tap/tubefold

or download TubeFold.zip directly

Why TubeFold

Your subscription, not an API

Runs your local codex or claude CLI under the hood. The model never sees an API key — it works off the plan you already pay for.

Native macOS app

Notarized Developer ID build with an embedded Python backend. Nothing to configure beyond pointing it at your CLI.

Clean Markdown out

Transcript in, structured Markdown summary out — ready to paste into notes, docs, or your knowledge base.

Updates itself

Built-in Sparkle auto-update keeps you on the latest version. Install once and forget about it.

How it works

  1. Paste a YouTube URL. video ID → metadata → transcript
  2. TubeFold pulls the transcript and wraps it in a summary prompt.
  3. Your CLI model runs locally — TubeFold takes only the final message.
  4. You get a Markdown summary to save or share.
